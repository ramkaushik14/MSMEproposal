Curriculum Modules

1
Foundations of GenAI
12 Hours
  • History and evolution of AI to Generative AI
  • Understanding ML, Deep Learning, and GenAI architectures
  • Neural networks, transformers, and attention mechanisms
  • Major GenAI models: GPT, BERT, Claude, Gemini, LLaMA
  • Ethics, bias, and responsible AI usage
  • Development environment setup and API configuration
2
Prompt Engineering Mastery
10 Hours
  • Basic and advanced prompt structures
  • Chain-of-thought reasoning and few-shot learning
  • Role-playing techniques and prompt patterns
  • Debugging and optimization strategies
  • Project: Building a comprehensive prompt library
3
Working with LLM APIs
12 Hours
  • REST APIs fundamentals and authentication
  • OpenAI API: Chat completions, embeddings, function calling
  • Anthropic Claude API: Messages and streaming responses
  • Google Gemini API and multi-provider integration
  • Error handling, rate limits, and cost optimization
  • Project: Multi-provider intelligent chatbot
4
RAG Systems & Vector Databases
10 Hours
  • Understanding embeddings and semantic search
  • Vector databases: Pinecone, Chroma, Qdrant
  • Building RAG pipelines: Document loading, chunking, retrieval
  • Advanced RAG: Hybrid search, reranking, metadata filtering
  • Project: Intelligent document Q&A system
5
LangChain & AI Frameworks
8 Hours
  • LangChain architecture and components
  • Building chains, agents, and memory management
  • Tool integration and custom components
  • Alternative frameworks: LlamaIndex, Haystack
  • Project: AI agent with tool-calling capabilities
6
Image & Multimodal AI
8 Hours
  • Text-to-image generation: DALL-E, Midjourney, Stable Diffusion
  • Image-to-text: Vision models and OCR
  • Audio generation and speech-to-text processing
  • Multimodal models: GPT-4V, Claude with vision
  • Project: Complete multimodal AI application

Capstone Project

10 Hours dedicated to building a complete, production-ready interactive GenAI application. Students will integrate RAG systems, API calls, user interfaces, and deployment strategies to create industry-standard projects including AI tutors, content generators, research assistants, or creative writing tools.

Program Outcomes

  • Build production-ready GenAI applications from scratch
  • Master prompt engineering for optimal AI interactions
  • Implement RAG systems with vector databases
  • Integrate multiple LLM APIs and frameworks
  • Create multimodal AI solutions
  • Deploy and maintain AI applications professionally
